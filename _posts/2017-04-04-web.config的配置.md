## web.config的配置

如果用connectionStrings元素，请使用下面这种形式：
   
```    
    <connectionStrings>
      <add name="" connectionString=""/>
    </connectionStrings>

```
如果用appSettings元素，请使用下面这种形式：

```
    <appSettings>
      <add key="" value=""/>
    </appSettings>
```
